# LineLogic

Phone-first can line troubleshooting app.

## Publish on GitHub Pages
1. Create a public GitHub repo named `linelogic`.
2. Upload these files into the repo.
3. Go to Settings > Pages.
4. Set Source to Deploy from a branch.
5. Branch: main. Folder: / root.
6. Save.
7. Open the GitHub Pages link on iPhone Safari.
8. Share > Add to Home Screen.
